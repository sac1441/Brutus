# Brutus 🎮

A Unity-based hypercasual game project focused on responsive controls, smooth gameplay, and mobile-first design.

## 🚀 Features
* 2D Character Controller (Tarodev-based)
* Touch + Keyboard input support
* Smooth jump mechanics (variable height + fast fall)
* Collectibles system (Eggs 🥚 & Flowers 🌸)
* Scene reset + ad trigger system
* Optimized for mobile gameplay
  
## 🛠️ Built With
* Unity (6000.x)
* C#
* Universal Render Pipeline (URP)

## 📂 Project Structure
Assets/ → Game assets, scripts, prefabs
Packages/ → Unity packages
ProjectSettings/ → Project configuration

## ▶️ How to Run
1. Open project in Unity Hub
2. Select correct Unity version
3. Open main scene
4. Press Play

## 🎮 Controls

| Action | Input            |
| ------ | ---------------- |
| Move   | Arrow Keys / A-D |
| Jump   | Spacebar / Touch |

## 📱 Platform
* Android (Primary)
* PC (Testing)
  
## 📌 Notes

* Unity Remote supported (for mobile testing)
* Uses ScriptableStats for tuning player behavior
* Optimized for hypercasual gameplay loop
